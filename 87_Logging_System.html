Logging System

A Logging System records application events, errors, warnings, and user activities. Logs help developers monitor applications, troubleshoot issues, and improve system reliability.

Log Levels

Level Description
INFO General application information.
DEBUG Detailed debugging information.
WARN Potential issues or warnings.
ERROR Application errors that need attention.
FATAL Critical system failures.

Sample Log Output

2026-06-05 10:00:15 INFO  User logged in
2026-06-05 10:05:30 WARN  High memory usage detected
2026-06-05 10:08:12 ERROR Database connection failed

Node.js Logging Example

const fs = require('fs');

function log(message) {
    const entry = `${new Date().toISOString()} - ${message}\n`;
    fs.appendFileSync('app.log', entry);
}

log('Application started');
