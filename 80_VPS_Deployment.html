VPS Deployment Guide

Step 1: Connect to VPS

Use SSH to connect to your VPS server:

ssh username@your-server-ip

Step 2: Update Server

sudo apt update
sudo apt upgrade -y

Step 3: Install Nginx

sudo apt install nginx -y
sudo systemctl start nginx
sudo systemctl enable nginx

Step 4: Deploy Website Files

Upload your project files to:

/var/www/html

Step 5: Configure Nginx

server {
    listen 80;
    server_name yourdomain.com;

    root /var/www/html;
    index index.html;
}

Step 6: Restart Nginx

sudo nginx -t
sudo systemctl restart nginx

Step 7: Enable HTTPS (Optional)

sudo apt install certbot python3-certbot-nginx -y
sudo certbot --nginx

Deployment Complete!

Your website should now be accessible through your VPS IP address or domain name.
